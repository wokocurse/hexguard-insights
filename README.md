## How can I deploy this project?

You can deploy HexGuard to any modern hosting provider that supports static sites or Node.js applications.

To create a production build:

```sh
npm run build
```

To preview the production build locally:

```sh
npm run preview
```

Deploy the generated build output to your preferred hosting provider (e.g., Vercel, Netlify, AWS, DigitalOcean, Cloudflare Pages).
